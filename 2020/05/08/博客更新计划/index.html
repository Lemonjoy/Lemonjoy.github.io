博客更新计划

发表于 2020-05-08 | 分类于 Python Study Notes | | 阅读次数:

近期对博客进行调整：

  1. 加入置顶权重，而不是仅仅依靠时间线来展示顺序。
  2. 由于最近忙于复试，很多题解没更新也没时间进行整理，打算复试后抽出时间进行整理归纳。
  3. 之后应该还会写一个爬虫，用来辅助更新LeetCode的readme，应该不是很难得一个脚本，最近有点懒，抽空写完并把其过程记录下来。

置顶权重修改

这可能是一项很大的工程，涉及到所有不是题解的文章，并为他们添加权重，以便于其显示在所有题解之前。

所以可能会写一个简单的脚本进行辅助整理。（然后就又有了可以水的素材，嘻嘻）

题解的更新

刷题断断续续已经要接近400题了，题解什么的却依旧保持在一百题的位置，实在是有点拖了，复试完应该就有足够的时间来进行总结这段时间所刷的题了，我深切的知道，总结比做题重要的多，现在目前看来虽然刷的很快，但自我的提升真的很有限，周赛的hard题还是做不出来，做题的速度也还是很慢，这充分证明了一切。

LeetCode爬虫

这其实是一个很早就想写的爬虫了，毕竟自己更新readme实在是太傻了点，虽然之前写个半自动的生成脚本，提升了不少效率，但是并不能自动获取标签是一个很大的问题，毕竟写标签也是个很耗时的工作。

懒惰有时候真的就是动力，至少在这件事上确实如此。

尾巴

最近的心情好了很多，之前主要是复试搞得整个人都很焦虑，近两天不断传来好消息，也能睡个安稳觉了。

我真的太想念书了，毕竟啊，人菜就要多读书。

努力提升自己永远都不会出错，加油！
